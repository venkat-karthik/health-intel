# Health Intelligence Platform

AI-powered health surveillance system for India with real-time disease predictions and interactive mapping.

## Features
- Interactive India map with state-wise health data
- AI chatbot powered by Perplexity
- Real-time disease predictions
- Comprehensive health analytics

## Tech Stack
- React + TypeScript + Vite
- Express.js backend
- Perplexity AI integration
- Tailwind CSS + shadcn/ui
